二维图形变换基本原理

变换的特点

一、图形变换了，但原图形的连边规则没有改变

二、图形的变换，是顶点位置的改变决定的

变换图形就是要变换图形的几何关系，即改变顶点的坐标；同时，保持图形的原拓补关系不变

仿射变换

仿射变换（Affine Transformation 或 Affine Map）是一种二维坐标到二维坐标之间的线性变换

  1. 平直性：直线经过变换后依然是直线
  2. 平行性： 平行线依然是平行线，且直线上点的位置顺序不变

齐次坐标

  • 【简而言之，就是用三维向量表示二维向量， 或者一般而言，用一个n+1维的向量表示一个n 维向量的方法称为齐次坐标表示法】

在二维平面内，我们是用一对坐标值（x，y）来表示一个点在平面内的确切位置，或者说使用一个向量（x，y）来标定一个点的位置

加入变换前的点坐标为（x，y），变换后的点坐标为（x，y），这个变换过程可以写成如下矩阵形式：

$\begin{bmatrix}x^\star,y^\star\end{bmatrix} = \begin{bmatrix}x,y\end{bmatrix}\cdot M $

M 为变换矩阵，即：

对于向量（x，y，1），可以在几何意义上理解为是第三位为常数的平面上的一个二维向量

这种用三维向量表示二维向量， 或者一般而言，用一个n+1维的向量表示一个n 维向量的方法称为齐次坐标表示法

n 维向量的变换是在n + 1 维的空间进行的， 变换后的n 维结果是被反投回到感兴趣的特定的维空间内而得到的。

例如：

n维向量$\begin{pmatrix}p_1,p_2,…,p_n\end{pmatrix} $表示为$\begin{pmatrix}hp_1,hp_2,…,hp_n,h\end{pmatrix} $，其中h称为哑坐标。

普通坐标与其次坐标的关系为“一对多”：

普通坐标 X h $\Rightarrow$ 齐次坐标

齐次坐标 / h $\Rightarrow$ 普通坐标

当h = 1时产生的齐次坐标称为“规格化坐标”，因为前n个坐标就是普通坐标系下的n维坐标

为什么要采用齐次坐标？

在笛卡尔坐标系内，向量（x，y）是位于z=0的平面上的点；而向量（x，y，1）是位于z=1的等高平面上的点

对于图形来说，没有实质性差别，但是却给后面矩阵运算提供了可行性和方便性

如果二维图形不用齐次坐标表示呢？

即坐标用二维坐标表示，那么根据矩阵乘法的定义，只有第一个矩阵的列数和第二个矩阵的行数相等时才能相乘，在矩阵运算中将有诸多不便，所以用齐次坐标表示法，用三维变量表示二维变量，如此一来，矩阵都为3X3矩阵，计算方式将统一。

窗口和视区

一、窗口和视区

世界坐标系中要显示的区域（通常在观察坐标系内定义）称为窗口

窗口映射到显示器（设备）上的区域称为视区

窗口定义显示什么；视区定义在何处显示

世界坐标系中的一个窗口可以对应于多个视区

要将串口内的图形在视区中显示出来，必须经过将窗口到视区的变换处理，这种变换就是观察变换（ViewingTransformation）

二、观察变换

  • 变焦距效果

当窗口变小时，由于视区大小不变，就可以放大图形对象的某一部分，从而观察到较大的窗口时未显示的细节

当窗口变大时，由于视区大小不变，就可以“缩小图形”

类似于相加的变焦处理

  • 整体缩放效果

当窗口大小不变而视区大小发生变化时，得到整体放缩效果，这种放缩不改变观察对象的内容

  • 漫游效果

当把一个固定大小的窗口在一幅大图形上移动，视区不变，产生的效果就是漫游效果

总结

  • 向量是图形处理中很重要的工具
  • 坐标系是建立图形与数之间对应联系的参考系
  • 图形和变化是计算机图形学的基础内容，是图形显示过程中不可缺少的环节
  • 齐次坐标给二维图形的矩阵运算提供了可行性和方便性
  • 物体变换：物体变换使用同一个规则改变物体上所有的点，但是保证底层坐标系不变
  • 坐标变换：坐标变换按照原坐标系统定义了一个全新的坐标系统，然后在新坐标系下表示物体上所有的点
  • 复合坐标：也称组合变换，即多种基本变换的组合。组合变换的变换矩阵时几个单独变换矩阵的乘积，由于矩阵乘法不满足交换率，因此在进行复合变换时，需要注意矩阵相乘的顺序
